# /timeline — 查看/编辑案件大事记

> 适用于案件初始化之后的任何阶段。展示 workspace/timeline.md 中的时间线。

## 前置检查

1. 检查 `workspace/timeline.md` 是否存在。
   - 存在 → 展示内容。
   - 不存在 → 提示"时间线尚未生成。执行 /cross-verify 后自动生成。"。

## 无参数：查看时间线

直接展示 `workspace/timeline.md` 的完整内容。

输出格式：
```
━━━ 案件大事记 ━━━

{timeline.md 全文}
```

## 带参数：编辑时间线

律师可以在对话中提供修改意见：
- "补充一个事件：2025年3月15日，甲方代表现场视察"
- "修改第5条日期为2025年6月1日"
- "删除第8条"

Agent 按律师意见修改 `workspace/timeline.md`，保持原有格式。

修改后确认：
```
时间线已更新。修改内容：
- {具体修改项}
```
